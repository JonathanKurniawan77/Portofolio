# Dokumentasi Singkat Website Portfolio Command Center

## 1. Deskripsi Website

Website ini adalah website portofolio pribadi dengan konsep **Digital Command Center**. Website menampilkan profil Jonathan Kur sebagai mahasiswa Sistem Informasi yang tertarik pada web development, UI/UX, jaringan komputer, dan data. Konsep command center dipilih agar website terlihat lebih unik, modern, dan berbeda dari portofolio umum.

## 2. Target Pengguna

Target pengguna website ini adalah:

- Dosen pengampu mata kuliah
- Teman kuliah
- Rekan pengalaman
- Pihak yang ingin mengetahui profil, kemampuan, dan pengalaman pemilik website

## 3. Konsep Visual

Website menggunakan background utama hitam dengan profil dan logo bernuansa hitam, abu-abu gelap, dan aksen putih agar tetap modern namun tidak terlalu neon. Tampilan dibuat seperti dashboard pusat kendali digital dengan elemen:

- Command panel
- Education timeline
- Skill stack
- Experience vault
- Transmission/contact form

Konsep ini bertujuan membuat portofolio terlihat lebih kreatif namun tetap mudah digunakan.

## 4. Penjelasan UI/UX

### Konsistensi

Warna, tipografi, border radius, card, dan tombol dibuat konsisten di seluruh halaman.

### Navigasi

Navigasi ditempatkan di bagian atas dan menggunakan sticky header agar mudah diakses. Pada tampilan mobile, navigasi berubah menjadi tombol menu.

### Hierarki Visual

Bagian hero menggunakan judul besar untuk menarik perhatian. Setiap section memiliki label, judul, dan isi yang tersusun jelas.

### Responsivitas

Layout website menggunakan CSS Grid, Flexbox, dan media query sehingga dapat menyesuaikan tampilan desktop, tablet, dan mobile.

### Feedback Pengguna

Website memberikan feedback melalui form kontak dan tab skill interaktif. Hal ini membuat pengguna merasa aksi yang dilakukan mendapatkan respons.

### Affordance

Tombol, link, menu, dan card dibuat memiliki gaya visual yang jelas sehingga pengguna dapat memahami elemen mana yang dapat diklik.

## 5. Teknologi

- HTML5
- CSS3
- JavaScript
- Google Fonts

## 6. Kelebihan Website

- Tampilan lebih unik dibanding portofolio standar
- Responsif
- Navigasi jelas
- Ada interaksi sederhana
- Struktur kode rapi
- Siap dihosting di GitHub Pages, Netlify, atau Vercel

## 7. Kesimpulan

Website ini sudah memenuhi ketentuan tugas karena memiliki tema bebas, memperhatikan prinsip dasar UI/UX, responsif, memiliki struktur kode rapi, dan siap diunggah ke GitHub serta dihosting secara online.
