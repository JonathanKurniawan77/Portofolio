# Portfolio Jonathan Kurniawan

Website ini adalah website portofolio . Website menampilkan profil saya sebagai mahasiswa Sistem Informasi yang tertarik pada web development, UI/UX, jaringan komputer, dan data. Website ini saya buat dengan tampilan modern bernuansa hitam, dengan navigasi yang jelas, layout yang rapi, serta responsif.

## Konsep Desain

Tema utama website ini adalah **pusat kendali digital**. Konsep ini dipilih agar berbeda dari portofolio umum yang biasanya hanya menggunakan layout standar foto, biodata, dan card biasa. Website ini menggunakan elemen visual seperti command panel, education timeline, skill stack, dan experience vault.

## Fitur Website

- Panel profil interaktif
- Navigasi sticky dan responsif
- Skill stack dengan tab interaktif
- Experience vault menggunakan layout bento/grid
- Form kontak dengan feedback JavaScript
- Desain responsif untuk desktop, tablet, dan mobile
- Struktur file rapi dan siap upload ke GitHub

## Prinsip UI/UX yang Diterapkan

### 1. Konsistensi Desain

Website menggunakan sistem warna gelap dengan aksen hitam, abu-abu gelap, dan aksen putih. Elemen seperti tombol, card, badge, dan panel dibuat dengan gaya visual yang konsisten.

### 2. Navigasi yang Jelas

Navigasi berada di bagian atas dan tetap terlihat saat halaman di-scroll. Pada layar kecil, navigasi berubah menjadi menu yang dapat dibuka dan ditutup.

### 3. Hierarki Visual

Judul dibuat besar dan kuat, sedangkan informasi pendukung menggunakan ukuran lebih kecil. Hal ini membantu pengguna memahami prioritas informasi.

### 4. Feedback

Form kontak memberikan pesan feedback setelah tombol diklik. Tab skill juga berubah ketika pengguna memilih kategori tertentu.

### 5. Responsivitas

Website menggunakan CSS Grid, Flexbox, dan media query agar tampilan tetap rapi di desktop, tablet, dan mobile.
